User Journey Map: Message Rediscovery

Persona: Priya Sharma (Product Manager)

1

Trigger

In a meeting, needs to reference a link shared by colleague in group chat 3 days ago

"Where was that design doc link?"

2

Attempted Recall

Tries to remember: Which group? Who sent it? What day? Any keywords?

⚠️ Memory fails — too many chats, too much context
3

Current Behavior

Opens WhatsApp search → tries "design" → 200+ results. Scrolls manually through group

⏰ 5-10 min wasted
4

Friction & Breakdown

PAIN POINTS:

  • • Can't filter by media type
  • • No time-based navigation
  • • Scroll is endless & disorienting
  • • Meeting is waiting
5

Opportunity Moments

✨ When search initiated
✨ When entering a chat
✨ During scroll fatigue

Critical Insight

Users fail at rediscovery NOT because they don't try, but because WhatsApp provides no contextual, semantic, or temporal tools to bridge the gap between "I need something" and "Here it is"
